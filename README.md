# 📚 SQL - Análise de Livros

Projeto desenvolvido durante o bootcamp de Análise de Dados da TripleTen.

## Objetivo

Analisar dados de corridas de táxi em Chicago utilizando SQL e Python para responder perguntas de negócio e validar uma hipótese estatística sobre o impacto das condições climáticas na duração das viagens.

## Tecnologias utilizadas

- Python
- Pandas
- SciPy
- SQL
- Jupyter Notebook

## Estrutura do projeto

## Principais análises

- Empresas de táxi com maior número de viagens
- Bairros mais utilizados
- Teste de hipótese sobre o impacto do clima na duração das corridas
