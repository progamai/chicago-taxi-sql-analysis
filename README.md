# 🚕 Chicago Taxi Analysis with SQL

Projeto desenvolvido durante o Bootcamp de Análise de Dados da TripleTen para analisar dados de corridas de táxi em Chicago utilizando SQL, Python e testes estatísticos.

---

# 📌 Objetivo

Analisar dados de corridas de táxi em Chicago para responder perguntas de negócio e verificar, por meio de testes estatísticos, se as condições climáticas influenciam a duração das viagens.

---

# 📖 Descrição do Projeto

Neste projeto foram utilizadas consultas SQL e análise de dados em Python para explorar informações sobre empresas de táxi, bairros e corridas realizadas em Chicago.

Além da análise exploratória, foi realizado um teste de hipótese para verificar se dias chuvosos apresentam diferenças significativas na duração das viagens em comparação aos dias sem chuva.

---

# 🔎 Metodologia

O projeto foi desenvolvido seguindo as seguintes etapas:

- Importação dos dados;
- Análise exploratória;
- Consultas SQL;
- Identificação das empresas com maior número de viagens;
- Identificação dos bairros mais utilizados;
- Teste estatístico da hipótese sobre condições climáticas;
- Interpretação dos resultados.

---

# 📊 Resultados

A análise permitiu:

- Identificar as empresas de táxi com maior volume de corridas;
- Identificar os bairros mais movimentados;
- Avaliar o impacto das condições climáticas na duração das viagens;
- Gerar recomendações baseadas em evidências.

# ▶️ Como executar

Clone o repositório:

```bash
git clone https://github.com/progamai/chicago-taxi-sql-analysis.git
```

Entre na pasta:

```bash
cd chicago-taxi-sql-analysis
```

Instale as dependências:

```bash
pip install -r requirements.txt
```

Abra o notebook:

```bash
jupyter notebook
```
